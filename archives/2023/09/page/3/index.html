#复数

复数的概念

形如：$z=x+yi$
其中$x,y∈R$
#虚数 单位i的特点：$i^2=-1$

记$x=Re(z),y=Im(z)$
复数相等：
$x1=x2,y1=y2$
共轭：$x_{1}=x_{2},y_{1}=-y_{2}$
$x=0,y\neq0$时，为纯虚数
$x\neq0,y=0$时，看作实数

复数四则运算

记$A=x_{1}+y_{1}i, B=x_{2}+y_{2}i$
和：$A+B=(x_{1}+x_{2})+(y_{1}+y_{2})i$
差：$A-B=(x_{1}-x_{2})+(y_{1}-y_{2})i$
积：$A\cdot B=(x_{1}x_{2}-y_{1}y_{2})+(x_{1}x_{2}+y_{1}y_{2})i$
商：$A\div B=\frac{(x_{1}x_{2}+y_{1}y_{2})}{x_{2}^2+y_{2}^2}+i\frac{x_{2}y_{1}-x_{1}y_{2}}{x_{2}^2+y_{2}^2}$

复数的 #模 与 #辐角

复数的几何表示

复数可以与复平面上的点一一对应
$|z|$：z的模
$|z|=r=\sqrt{x^2+y^2}$
$|x|<|z|, |y|<|z|$
$|z|\le|x|+|y|$
$|z|\ge0$
$z\cdot\overline{z}=|z|^2=|z^2|=|\overline{z}|^2=|\overline{z}^2|$

于Windows上部署ChatGLM2-6B 以及其WebUI 人工智能
ChatGLM2-6B LLM Python - 2023-09-02 - 2023/09/02/Blog/于Windows上部署ChatGLM2-6B 以及其WebUI/

寻思要给自己搞一个能上网的GLM2-6B，就在GLM官网找有没有友链，正好找到了这么一条：ChatGLM-6B-Engineering，于是就打算在本地部署一下（我是比较讨厌语言模型不在本地的那种）。坑有点多，于是用几乎是0基础的方式写了这么一篇教程。
本篇文章使用venv来创建虚拟环境，conda创建环境的可以找别人了。

0x0 拉代码

到这个工程的仓库拉下代码，解压到一个文件夹。
创建一个命令行窗口，然后cd到你的目录。
rem Go to your working dir. 
cd /D D:\path\to\your\workspace

0x1 创建环境

然后创建python虚拟环境（有些人极度讨厌虚拟环境可以跳过，此处使用venv而不是conda是因为我比较讨厌conda）：
rem Run python to create venv. 
python -m venv .\venv

确保缓存的盘足够，如果不能的话，就会报pip没有足够的空间安装，你需要这么做：
rem Set the cache dir. 
set TMPDIR=D:\your\cache\dir

然后再在同一窗口安装执行：
pip安装依赖：
pip install -r requirements.txt
